To make a new database with the current relational schema, create a new project in Supabase. Then use through terminal:

# Login to Supabase
npx supabase login


npx supabase link --project-ref project_id

#This is to push current changes into your project
npx supabase db push

#This is just your usual workflow, ig you'll only need it once or twice

#pull from main
git checkout main
git pull origin main

#just in case backend is changed
npx supabase db push

#new branch, replace branch name
git checkout -b new_branch

#add changes
